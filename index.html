Python Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Basic Syntax

# Hello World
        print("Hello, World!")
        
        # Variables
        x = 5
        y = "John"
        
        # Functions
        def my_function():
            print("Hello from a function")
        
        my_function()

Java Introduction

Java is a class-based, object-oriented programming language designed to have few implementation dependencies.

Basic Syntax

// Hello World
        public class Main {
            public static void main(String[] args) {
                System.out.println("Hello World");
            }
        }
        
        // Variables
        int myNum = 5;
        String name = "John";
        
        // Methods
        static void myMethod() {
            System.out.println("Hello from a method");
        }

JavaScript Introduction

JavaScript is a lightweight, interpreted programming language with first-class functions.

Basic Syntax

// Hello World
        console.log("Hello, World!");
        
        // Variables
        let x = 5;
        const name = "John";
        
        // Functions
        function myFunction() {
            console.log("Hello from a function");
        }
        
        // Arrow function
        const myArrowFunc = () => {
            console.log("Hello from arrow function");
        };

C# Introduction

C# is a modern, object-oriented programming language developed by Microsoft.

Basic Syntax

// Hello World
        using System;
        
        class Program {
            static void Main() {
                Console.WriteLine("Hello World");
            }
        }
        
        // Variables
        int myNum = 5;
        string name = "John";
        
        // Methods
        static void MyMethod() {
            Console.WriteLine("Hello from a method");
        }
